Welcome! This package contains a complete Python analysis pipeline for testing
whether the MOND acceleration scale (a₀) shows scale-dependence.

================================================================================
QUICK START (TL;DR)
================================================================================

1. Create directories:
   - src/
   - config/
   - data/raw_sparc/
   - output/
   - logs/

2. Copy Python files to src/:
   - __init__.py, sparc_loader.py, rar_fitting.py, radial_analysis.py
   - statistical_tests.py, visualization.py, main.py

3. Copy config file:
   - analysis_config.yaml → config/

4. Copy main scripts to project root:
   - run_analysis.py, run_analysis.bat, requirements.txt

5. Copy SPARC data to data/raw_sparc/:
   - Table1.mrt, Table2.mrt, rar_all_data.txt, sparc_mass_models/

6. Install dependencies:
   pip install -r requirements.txt

7. Run:
   python run_analysis.py

8. Check results in:
   - output/tables/interpretation.json
   - output/plots/radial_rar_analysis.png

================================================================================
WHAT THIS DOES
================================================================================

1. Loads 175 SPARC galaxies with high-quality rotation curves
2. Fits the McGaugh MOND formula (RAR) to inner and outer radii
3. Compares a₀(inner) vs a₀(outer):
   - If inner > outer → Supports SDH+ (quantum RG-running)
   - If inner ≈ outer → Supports ΛCDM (universal a₀)
4. Computes statistical significance across ensemble
5. Generates diagnostic plots and interpretation
